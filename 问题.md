6、请问在Oracle中‘create table xxx  nologging as select * from abc’的速度要快于'create table xxx as select * from abc';不过它有什么不好的地方。



答案：

nologging:表示不需要写redo日志或者写入的redo日志远远小于正常的表的插入数据；对于归档数据库来说，可能会导致数据恢复的丢失
