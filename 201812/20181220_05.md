## 上传文件解压文件

```
# cd /software
# tar -zxvf mysql-5.6.41-linux-glibc2.12-x86_64.tar.gz 
# mkdir -p /opt/mysql-5.6/
# cp -R mysql-5.6.41-linux-glibc2.12-x86_64/* /opt/mysql-5.6/

```

##  创建mysql用户

```
# groupadd mysql 
# useradd -r -g mysql -s /bin/false mysql
```



## 授权

```
# chown -R mysql:mysql /opt/mysql-5.6/
# chown -R mysql:mysql /opt/mysql-5.6
```
