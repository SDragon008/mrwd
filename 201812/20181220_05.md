## 创建用户

```
# groupadd mysql
# useradd -m -s /bin/bash/ -d /home/mysql -g mysql mysql
# passwd mysql
```





## 上传文件解压文件

```
# cd /software
# tar -zxvf mysql-5.6.41-linux-glibc2.12-x86_64.tar.gz 
# mkdir -p /opt/mysql-5.6/
# cp -R mysql-5.6.41-linux-glibc2.12-x86_64/* /home/mysql

```



## 创建目录

```
# cd /home/mysql/
# mkdir logs
```



## 授权

```
# chown -R mysql:mysql /home/mysql
```
